Morse Code

The Morse Code is a method of transmitting text using sequences of short and long signals.
Using dots and dashes.



 A brief overview of what Morse code is.

 Developed in the 1830s by Samuel Morse and Alfred Vail as a way to communicate source of information. Morse code is a telecommnuication method that encodes text characters( letters ,numbers,and punctiuaction ) into standardized sequences of dots and dashes. 

 The historical context of why morse code was created.
 Morse code was created to overcome the physical limitations of early electrical telegraphs and develop newly electric telegraphs. Morse code relied on a binary like system of on - off signals over long distances. 

A = .- 
B = -....
1 = .----
 
 A dash is a longer signal and a dot is a short signal.

An explanation of how the morse code system works, including the use of dots and dashes:

Telegraph Machines- which produce clicking sounds. Morse code is transmitted using on - off keying over any medium that carries signals. The motion of tappping a transmitter, the operator alternates between short electrical and light pulses(dots) and long pulses(dashes).
Morse Code is a communication system that translates text into sequences of short pulses(dots) and long(dashes)
A dash is exactly three times the duration of a dot, and silence gaps of varying lengths (1,3 & 7) units separate indivdual signals, letters and words to ensure the message is readable.
